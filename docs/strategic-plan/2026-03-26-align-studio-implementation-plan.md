**Key App Features for MVP:**
1. **Calendar & Booking** — View schedule by day/week; book specific reformer spot (1 of 4) or mat class spot (1 of 6); waitlist for full classes
2. **User Profiles** — Name, contact, photo, medical form (PAR-Q), booking history
3. **Medical Form (PAR-Q)** — In-app form completed on first booking; stored securely; Katie can review before class
4. **Stripe Payments** — Buy class packs, memberships, single sessions; auto-renewal for memberships; Apple Pay and Google Pay support
5. **Push Notifications** — Class reminders (24hr and 1hr before), cancellation alerts, new class announcements
6. **Class Descriptions** — Each class type with description, difficulty level, what to bring, what to expect
7. **Schedule Display** — Weekly timetable view; filter by class type
